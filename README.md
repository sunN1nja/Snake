# Snake

A simple Python and pygame game with high score saving.

## Run

```powershell
python main.py
```

If pygame is not installed:

```powershell
python -m pip install -r requirements.txt
```

## Controls

- Arrow keys or WASD - move
- Space - pause
- R - restart after losing
- Esc - quit

The high score is saved in `high_score.json` next to the game.
